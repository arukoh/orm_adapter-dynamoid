OrmAdapter::Dynamoid

Adds dynamoid adapter to orm_adapter which provides a single point of entry for using basic features of popular ruby ORMs.

Installation

Add this line to your application's Gemfile:

gem 'orm_adapter-dynamoid'

And then execute:

$ bundle

Or install it yourself as:

$ gem install orm_adapter-dynamoid
